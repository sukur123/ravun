# Contributing to Ravun

We welcome contributions! If you’re interested in helping shape Ravun’s future, you can contribute in the following ways:

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Create a new branch** for your feature or fix.
3. **Make your changes**, ensuring to follow the style guidelines.
4. **Submit a pull request** for review.

## Areas for Contribution

We are particularly interested in contributions for the following areas:
- Language design and syntax improvements.
- Compiler optimizations and bug fixes.
- Writing documentation and tutorials.
- Developing tools for Ravun, including IDE support and testing frameworks.
- Community engagement to help spread Ravun’s reach.

## Code of Conduct

Please make sure to follow the [Code of Conduct](CODE_OF_CONDUCT.md) while contributing. This ensures that we maintain a respectful and productive environment for all contributors.
